打卡外挂v1.0正式版（2014-01-08）
功能列表：
===============================================
1.支持工作日上下班四次打卡，非工作日（周六、周日）不打卡。
2.打卡时间可配置。
3.支持节假日以及工作日动态配置。（节假日不打卡，工作日打卡）
4.打卡时间随机生成。

安装使用说明：
===============================================
1.本软件无需安装，解压直接运行punch.bat即可。
2.运行依赖java运行环境（jdk1.6及以上版本）。
3.本软件依赖电脑时间，运行前请确保电脑时间和打卡服务器时间一致，否则可能造成打卡时间不正确。
4.确保配置文件（conf/config.properties）中的用户名（username）、密码（password）和run.com中登录的用户名密码保持一致。
5.软件运行后请确保不要将软件窗口关闭，窗口关闭后软件会停止运行。
6.配置文件修改后，务必重启此软件，否则配置不生效。
7.本软件默认周末不打卡，如需周末打卡可以在workingdays属性中配置。
8.配置文件格式要求UNIX UTF-8格式，请修改后注意检查。

FAQ：
===============================================
1.运行punch.bat文件出现命令无效。
请正确设置JAVA_HOME和PATH，确保java命令能正确执行，可以在cmd窗口中，运行java - version，命令来检查java环境。
2.打卡时发现登录失败
请正确配置config.properties中的用户名和密码，注意去掉多余空格。
3.正确配置用户名和密码后发现还是不能正常打卡。
由于公司用户名为中文，请确认配置文件格式是否是UNIX UTF-8。
4.如何配置节假日，工作日（周六、日）
配置文件中增加vacationDays或workingDays，多个日期中间请用半角逗号隔开。
